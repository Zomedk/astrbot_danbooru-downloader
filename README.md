# AstrBot Danbooru美图插件

一个用于从Danbooru获取随机美图的AstrBot插件。

## 功能特性

- 🎭 支持14个热门二次元角色
- 🖼️ 随机获取角色图片
- 📤 发送后自动删除本地文件
- 🔞 只获取全年龄内容（rating:g）
- 💬 QQ聊天命令交互

## 支持的角色

- 明日方舟：管理员、佩丽卡、艾尔黛拉、陈千语
- 原神：胡桃、雷电将军、神里绫华
- Re:Zero：拉姆、雷姆
- 电锯人：玛奇玛、帕瓦
- 其他：02、阿尼亚、约尔

## 安装方法

1. 在AstrBot插件管理页面上传插件压缩包
2. 配置Danbooru用户名和API Key

## 配置说明

在插件配置页面填写：
- **Danbooru用户名**：你的Danbooru账户用户名
- **API Key**：在Danbooru账户设置中生成的API Key

## 命令用法

| 命令 | 说明 | 示例 |
|------|------|------|
| `美图 <角色名>` | 获取随机一张角色美图 | `美图 管理员` |
| `美图角色` | 查看支持的角色列表 | `美图角色` |

## 使用示例

```
用户: 美图 管理员
机器人: 正在获取 [管理员] 的美图...
机器人: [发送图片]

用户: 美图角色
机器人: 支持的角色列表：
管理员
佩丽卡
艾尔黛拉
...
```
