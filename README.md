# AstrBot Danbooru图片下载器插件

一个用于下载Danbooru图片的AstrBot插件，支持热门二次元角色图片下载。

## 功能特性

- 🎭 支持14个热门二次元角色
- 📦 自动按角色创建文件夹
- ⚡ 异步下载，速度更快
- 🖼️ 只下载图片文件（过滤视频）
- 💬 QQ聊天命令交互

## 支持的角色

- 明日方舟：管理员、佩丽卡、艾尔黛拉、陈千语
- 原神：胡桃、雷电将军、神里绫华
- Re:Zero：拉姆、雷姆
- 电锯人：玛奇玛、帕瓦
- 其他：02、阿尼亚、约尔

## 安装方法

1. 在AstrBot插件管理页面上传插件压缩包
2. 配置Danbooru用户名和API Key
3. 在QQ聊天中使用命令

## 配置说明

在插件配置页面填写：
- **Danbooru用户名**：你的Danbooru账户用户名
- **API Key**：在Danbooru账户设置中生成的API Key

## 命令用法

在QQ聊天中发送以下命令：

| 命令 | 说明 | 示例 |
|------|------|------|
| `danbooru下载 <角色名> [数量]` | 下载指定角色的图片 | `danbooru下载 管理员 50` |
| `danbooru状态` | 查看当前下载状态 | `danbooru状态` |
| `danbooru角色` | 查看支持的角色列表 | `danbooru角色` |

## 使用示例

```
用户: danbooru下载 管理员 50
机器人: 开始下载角色 [管理员] 的图片，目标数量：50
机器人: 找到 50 张图片，开始下载...
机器人: 角色 [管理员] 下载完成！成功下载: 50/50 张

用户: danbooru角色
机器人: 支持的角色列表：
管理员
佩丽卡
艾尔黛拉
...

用户: danbooru状态
机器人: 当前无下载任务
```

## 注意事项

- 请确保你的Danbooru账户有足够的权限
- 建议使用等级较高的账户以获得更好的API访问权限
- 下载的图片将保存在插件目录的downloads文件夹中
